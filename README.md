# CMS — Enterprise Headless Content Platform

A clean, modular, scalable Headless CMS built with .NET 10.

## Features

- Modular Architecture (Clean Architecture + DDD + CQRS)
- Redis Caching
- RabbitMQ Messaging (MassTransit)
- Elasticsearch 8 Search Engine
- Media Management
- JWT Authentication + Roles
- Webhooks with retry & signature
- Docker Compose for full stack

## Modules

- Content (Types, Items, Versioning)
- Media (Upload, Resize)
- Search (Indexing, Querying)
- Identity (Users, Roles)
- Webhooks (Subscriptions, Delivery)

## Architecture

- API Layer  
- Application Layer  
- Domain Layer  
- Infrastructure Layer  
- Worker Service  

## Deployment

- Docker Compose  
- Nginx Reverse Proxy  
- SSL Ready  
- Health Checks
